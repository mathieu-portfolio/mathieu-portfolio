# Mathieu — C++ Systems Engineer

I build performance-oriented C++ projects focused on simulation, runtime systems, and low-level architecture. My work sits at the intersection of data-oriented design, deterministic simulation, and measurable execution — systems that are explicit in their ownership, honest about their invariants, and easy to reason about at scale.

I care about concurrency, scheduling, profiling, and the kind of reproducibility that makes benchmarks mean something. Whether it's CPU/GPU pipelines or memory layout decisions, I tend to work from first principles and measure as I go.

## Approach

I prefer focused implementations over feature-heavy designs, explicit control over convenient abstractions, and systems that can be measured rather than assumed. Each project is intentionally scoped — the goal is to go deep on one technical problem, not to cover ground.
